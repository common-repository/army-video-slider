=== Description ===

The A.R.M.Y. VideoSlider Plugin allows you to easily add a video slider to your WordPress site, 
showcasing your favorite embedded videos in a dynamic way because this plugin allows you to 
insert multiple videos in a slider using iframes through Shortcodes.

=== Installation ===

1. Upload and install army-video-slider.zip

2. Activate the plugin through the 'Plugins' menu in WordPress.

=== Usage ===

Adding an A.R.M.Y. VideoSlider to your WordPress site is easy. Just follow these steps:

1. **Open a Page or Post:** Navigate to the page or post where you want to showcase the video slider.

2. **Use Text/HTML Mode:** While in the content editor, ensure that you are in the 'Text' or 'HTML' editing mode. This allows you to insert HTML code and shortcodes directly.

3. **Add the Shortcode:** Place the following shortcode in the desired location within the editor:

   Shortcode:

   [army_videoslider video_links="https://www.youtube.com/embed/NnTYmg4LmzI,https://www.youtube.com/embed/S8IA3qONp8c,https://www.youtube.com/embed/bmDXgBDF7lk"]

4. **Replace Video Links:** Replace the placeholders with the actual full links of the embedded videos you wish to display in the slider. Separate the links using commas.

5. **Save and Update:** After adding the shortcode and video links, save or update the page or post to apply the changes.

6. **View the Result:** Once the page is published or updated, you'll see the A.R.M.Y. VideoSlider with the provided links.

Enjoy enhancing your site with an engaging A.R.M.Y. VideoSlider!
